## 🌐 Test Website Deployment

This repository contains a simple test website created for the purpose of learning and demonstrating how to publish a website using GitHub Pages.

The project is not intended for production use and does not represent a complete or fully functional application. Instead, it serves as a practice environment to understand the process of:

* Creating a repository
* Uploading website files
* Configuring GitHub Pages
* Successfully deploying a live website

This test project helps build foundational knowledge in web hosting and version control using GitHub.
